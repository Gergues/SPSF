SP Helpers

Helpful commands to run SharePoint operations

Recipes

  • Activate Feature

    Activates the selected feature in the selected site or web.

  • Deactivate Feature

    Deactivates the selected feature in the selected site or web.

  • Uninstall Feature

    Uninstalls the selected feature.

  • Copy Appbin Content

    Runs stsadm -o copyappbincontent.

  • Execute Admin Service Jobs

    Runs stsadm -o execadmsvcjobs to run all pending deployment jobs.

  • Open stsadm Prompt

    Opens a prompt to enter stsadm commands manually.
