Simple BCS Model

Add a simple BCS model

Arguments

Name Description
Model Settings
Name Required string. Enter the name of the BCS Model.
Display Name Required string. Enter the display name of the BCS Model. This name is visible in dialogs to the user.
Estimated Item Count Required integer. Enter number of estimated count of elements in the data source.
Is Cached Required boolean. Check if the data should be cached.
Permissions
Users Enter all users which should have permission to this model, separated by ';', e.g. 'NT AUTHORITY\Authenticated Users' or 'STS|SecurityTokenService|http://sharepoint.microsoft.com/claims/2009/08/isauthenticated|true|http://www.w3.org/2001/XMLSchema#string'.
