Helpers

Contains recipes to speed up the development process

Recipes

  • Exclude item from SourceControl

    Removes the item from source control, e.g. import for manifest.xml and makecab.ddf to enable automatic generation of these files.

  • Run MSBuild Target

    Runs a target from an msbuild file.

  • Get AssemblyName

    Displays the full assembly name of the current selected project.

  • Get Installed SharePoint Version

    Displays the version of SharePoint on the local machine.

  • Open WSP solution file as .CAB

    Displays the contents of the wsp solution file.

  • Open in Browser

    Opens the selected item in browser.

  • Open SPSF Help

    Opens the SPSF help in the browser.

  • About SPSF

    Displays information about the current installed SPSF version.
