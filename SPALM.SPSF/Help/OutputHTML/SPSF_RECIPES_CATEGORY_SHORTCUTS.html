Shortcuts

Contains useful shortcuts

Recipes

  • Reload SharePoint Configuration

    Reloads the file SharePointConfiguration.xml with the contents of the local SharePoint farm (features, listtemplates, content types etc.).

  • Open Debugging Site in Browser

    Opens the site for debugging of the project in the browser.

  • Open Item in SharePoint Hive

    Opens the current selected item in SharePoint Hive.

  • Open Latest SharePoint Trace Log

    Opens the latest SharePoint Trace Log file e.g. to check deployment issues.

  • Open SharePoint Trace Log Folder

    Opens the folder with the SharePoint Trace Log files.

  • Open Central Administration in Browser

    Displays the Central Administration.

  • Open Solution Management in Browser

    Displays the web page "Solution Management" in SharePoint Central Administration.
